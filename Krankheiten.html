Krankheiten

Finden Sie unten Ihre Krankheit für schnelle RP-Anleitungen

INFORMATION

Bitte achten Sie darauf, dass jeder Patient nach allen Symptomen gefragt und kontrolliert wird. Überprüfen Sie (insbesondere bei schweren und ansteckenden Krankheiten), ob dieser Patient nicht falsch diagnostiziert werden soll. Holen Sie sich nach Möglichkeit eine zweite Meinung von einem Arzt ein oder fragen Sie einen Spezialisten danach. Wenn kein Spezialist verfügbar ist, senden Sie ihm ein Telegramm mit allen von Ihnen gesammelten Informationen und stellen Sie außerdem sicher, dass ein zweiter Arzt die gleiche Diagnose stellt. Eine Fehldiagnose kann zum Tod eines Patienten oder zu einem psychischen Trauma führen, das wir verhindern wollen. Auch leitende und leitende Ärzte sind in der Lage, bei Bedarf eine gute Zweitmeinung abzugeben.

Pest +
Beschreibung +

Heftiges Fieber, Lymphknotenschwellungen, oft schwarzer Ausschlag.

Behandlung +

Isolation, Ruhe, Kamillentee, hochprozentiger Alkohol zur Desinfektion. Meist tödlich.

Cholera +
Beschreibung +

Starker Durchfall, Erbrechen, Austrocknung.

Behandlung +

Kochsalzlösungen, Glukoselösungen, Rizinusöl. Überlebenschancen sehr gering bei starkem Verlauf.

Typhus +
Beschreibung +

Fieber, Ausschlag, Schwäche, Bauchschmerzen.

Behandlung +

Ruhe, Flüssigkeit, Fiebertonikum, Pfefferminztee. Todesfälle häufig.

Tuberkulose +
Beschreibung +

Chronischer Husten, Auswurf, Schwäche, Nachtschweiß.

Behandlung +

Ruhe an frischer Luft, Kamillentee, Chinin-Lösung. Chronisch verlaufend, Heilung selten.

Diphtherie +
Beschreibung +

Halsentzündung, grauer Belag, Atemnot.

Behandlung +

Diphtherie-Serum, Ruhe, warme Getränke. Bei schwerem Verlauf oft tödlich.

Masern +
Beschreibung +

Fieber, Hautausschlag, Husten, Augenentzündung.

Behandlung +

Kamillentee, Salbeitee, Fiebertonikum, Ruhe. Meist gutartig bei Kindern, Komplikationen möglich.

Keuchhusten +
Beschreibung +

Heftige Hustenanfälle, Erstickungsgefahr bei Kleinkindern.

Behandlung +

Bronchialelixir, Ruhe, Kamillentee. Risikopatienten oft gefährdet.

Grippe / Influenza +
Beschreibung +

Hohes Fieber, Gliederschmerzen, Husten, Schwäche.

Behandlung +

Aspirin, Antipyrin, Kamillentee, Ruhe. Normalerweise selbstlimitierend, Komplikationen möglich.

Pocken +
Beschreibung +

Fieber, eitriger Hautausschlag, Narbenbildung.

Behandlung +

Ruhe, Flüssigkeit, Kamillentee, Opiumelixir gegen Schmerzen. Hohe Sterblichkeit.

Wundfieber / Gangrän +
Beschreibung +

Entzündete Wunden, Eiter, Fäulnis, übler Geruch.

Behandlung +

Carbolsäure, Jodlösung, Senfpflaster, Arnika-Tinktur. Ohne Behandlung oft tödlich.

Durchfall / Cholera-artige Symptome +
Beschreibung +

Häufiger, wässriger Stuhl, Austrocknung.

Behandlung +

Glukoselösungen, Kochsalzlösung, Pfefferminztee, Rizinusöl. Schnell lebensbedrohlich.

Syphilis +
Beschreibung +

Stadien: Geschwür → Ausschlag → Organbefall.

Behandlung +

Quecksilberverbindungen, Ruhe. Frühstadium behandelbar, Spätstadium gefährlich.

Gonorrhoe +
Beschreibung +

Brennen beim Wasserlassen, eitriger Ausfluss.

Behandlung +

Quecksilberverbindungen, Salbeitee, Ruhe. Komplikationen möglich, chronisch verlaufend.

Anämie +
Beschreibung +

Schwäche, Blässe, Müdigkeit.

Behandlung +

Liebig’s Malt Extract, Ruhe, Chinin-Lösung. Chronisch, Heilung oft schwierig.

Malaria +
Beschreibung +

Fieberanfälle, Schüttelfrost, Schwäche.

Behandlung +

Chininpulver, Chinin-Lösung, Ruhe. Wiederkehrend, therapierbar mit Chinin.

Gelbsucht / Hepatitis +
Beschreibung +

Gelbfärbung von Haut und Augen, Schwäche.

Behandlung +

Ruhe, Kamillentee, leichte Kost. Heilung möglich, Verlauf oft langwierig.

Nierenentzündung +
Beschreibung +

Schmerzen im Rücken, Schwellungen, Fieber.

Behandlung +

Ruhe, Flüssigkeit, Kamillentee. Komplikationen häufig.

Ringelflechte +
Beschreibung +

Rundlicher, schuppender Ausschlag.

Behandlung +

Zinksalbe, Bienenwachs-Salbe, Hygiene. Meist harmlos, Behandlung nötig.

Ekzeme +
Beschreibung +

Juckende, gerötete Hautstellen.

Behandlung +

Zinksalbe, Bienenwachs-Salbe, Opiumsalbe bei starkem Juckreiz.

Frostbeulen +
Beschreibung +

Schmerzhafte, bläuliche Haut, oft Finger oder Zehen.

Behandlung +

Arnikasalbe, warme Umschläge, Mentholsalbe. Schmerzstillend, Heilung möglich.

Hitzeschlag / Sonnenstich +
Beschreibung +

Anstieg von Körpertemperatur (stark), Haut heiß und gerötet, Kopfschmerzen, Schwindel + Übelkeit

Behandlung +

Sofort in den Schatten bringen, Kleidung lockern und Bettruhe. Körper mit kalten Umschlägen oder Eis kühlen (Kopf, Nacken, Achseln). Bei Bewusstsein Wasser oder leichte Salzlösung geben. Kreislauf überwachen.

Melancholie / Depression +
Beschreibung +

Niedergeschlagenheit, Schlafstörungen, Appetitverlust.

Behandlung +

Bromide, Ruhe, Spaziergänge an frischer Luft, Frauengold. Unterstützend, Heilung schwer.

Wahnsinn / Epilepsie / Hysterie +
Beschreibung +

Krampfanfälle, Wutausbrüche, Verwirrtheit.

Behandlung +

Bromide, Ruhe, Mentholpastillen. Symptome linderbar, Heilung selten.
